Key points to grasp

1) Light has wave-like properties, and color is determined by wavelength

2) If a light source approaches an observer, the wavelength is perceived to be compressed (higher frequency) resulting in a blue shift

3) If moving away from an observer, the wavelength is elongated (lower frequency), and a shift to the red end of the spectrum is observed
